1. Kalıtım zincirine bakmak (en kesin yöntem)

Checked exceptions: Exception sınıfından türeyip RuntimeException’ın alt sınıfı olmayanlardır.

Unchecked exceptions: RuntimeException veya onun alt sınıflarıdır.

Örnek

Checked

IOException extends Exception  // checked

Unchecked

NullPointerException extends RuntimeException  // unchecked
Özet kural:
  • RuntimeException ve alt sınıfları → unchecked
  • Exception (ama RuntimeException değil) → checked
